# selenium-scraping

youtube-scrape : Python script uploads YouTube video data to [a Google Sheet](https://docs.google.com/spreadsheets/d/1sz65hZuFaZomwZrqBmEUzanQtCVj_drUJYkmSNuEqmM/edit#gid=1683412517).
